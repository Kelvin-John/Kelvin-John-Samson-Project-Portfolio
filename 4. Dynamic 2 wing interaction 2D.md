<h1>Dynamic 2 wing interaction 2D</h1>
- The primary goal of this simulation was to study the serodynamic interaction between different elemnts in an active 2 element FSAE wing.
- This was done using ANSYS Fluent and an overset dynamic mesh.
- Post processing was done using ANSYS CFD post which cannot process dead and active cells stacked hence the errors in the contour and animation.
