<h1>4.Dynamic 2 wing interaction 2D</h1>

- The primary goal of this simulation was to study the aerodynamic interaction between different elemnts in an active 2 element wing for the 2025 Hybrutos car.
- This was the core platorm used to design the control system for the weight tranfer based active aero for the 2026 Hybrutos car
- This was done using ANSYS Fluent and an overset dynamic mesh.
- Post processing was done using ANSYS CFD post which cannot process dead and active cells stacked hence the errors in the contour and animation.
- Tried using paraview but as I had exported only the case and data files I did not have sufficient ram to post process, subsequently event tried ensight but had the same export file type problem which is why CFD post was used.
